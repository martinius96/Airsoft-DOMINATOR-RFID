RFID Domination Timer

RFID enabled stopwatch for better experience from Airsoft games

Find Out More

We've got what you need!

Stopwatch equipped with LCD character display, AVR ATmega328P MCU, RFID reader NXP RC522, LEDs and piezo. Suitable for up to 4-day non-stop operation (MilSim, LARP). There are two versions available, 1.0 for 2 team firmware, 2.0 for 4 team firmware. There are minor changes in both PCB designs regarding number of diodes, service holes etc.

Advantages of the system

Sports

Can be used for Airsoft, Paintball or any sport

Scalable

Easily scalable for 2 up to 4 teams per game

PCB, modular design

PCB ready, design that allows to hotswap components without need to unsolder anything

Game modes

Domination (primary) or Countdown game mode available
